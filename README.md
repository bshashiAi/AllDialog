# Dialog Library - Professional Dialog Management for Java

[![JitPack](https://jitpack.io/v/bshashiAi/dialog-library.svg)](https://jitpack.io/#bshashiAi/dialog-library)
[![Java CI](https://github.com/bshashiAi/dialog-library/workflows/Java%20CI/badge.svg)](https://github.com/bshashiAi/dialog-library/actions)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

A powerful, feature-rich dialog management library for Java applications.

## Features

- ✅ Multiple dialog types (Info, Warning, Error, Success, Confirmation)
- ✅ Dialog queuing system
- ✅ History tracking
- ✅ Custom theming support
- ✅ Async dialog handling
- ✅ Timeout auto-dismiss
- ✅ Global and specific listeners
- ✅ Thread-safe implementation
- ✅ Extensible and customizable

## Installation

### Using JitPack (Maven)

Add JitPack repository to your `pom.xml`:

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
